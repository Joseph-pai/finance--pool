# FamilyPool 權限與邏輯重構計劃

## 變更目標
1. 實施三層角色權限 (Admin, Lake Manager, Member)。
2. 確保數據原子性與完整性（原子操作、自動餘額調整）。
3. 修復預測邏輯並同步 UI。

## 角色定義
- **Admin**: 全域管理權限。
- **Lake Manager**: 湖泊管理與審批權限。
- **Member**: 自己帳目管理，全家數據檢視。

## 主要變更內容
1. **資料庫**: 更新 RLS、角色與觸發器。
2. **Hook**: 更新 `useAuth` 提供權限屬性。
3. **頁面**: 更新 Income, Expenses, Requests 頁面的權限邏輯。

## 自動化邏輯
- 刪除已確認收入 -> 自動扣除餘額。
- 刪除已完成支出 -> 自動還原餘額。
- 批准調撥申請 -> 自動標記對應支出項目為已完成。
